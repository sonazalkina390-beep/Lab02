# NumberSystemConverter
### Программа для работы с системами счисления в Windows Forms:
- Конвертер десятичных чисел в двоичную, восьмеричную и шестнадцатеричную системы
- Калькулятор сложения двух двоичных чисел
- Визуализация двоичного представления числа в виде графических примитивов
### Входные данные:
- Задача 1: Десятичное число (целое, неотрицательное)
- Задача 2: Две строки, содержащие только символы '0' и '1' (двоичные числа)
- Задача 3: Десятичное число (целое, неотрицательное) для визуализации

### Как запустить через Visual Studio
- Склонируйте репозиторий
- Откройте файл NumberSystemConverter.sln (или Lab02.sln, в зависимости от названия решения)
- Запустите программу (Файл > Запуск или нажмите F5)

### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub. (Ссылку нужно будет вставить после создания репозитория)
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с проектом. Если нужно указать другую папку, в конце команды для клонирования нужно указать её название: git clone <ссылка на репозиторий> <папка>

### Пример использования
- Пользователь запускает приложение и видит главное меню с тремя кнопками.
- При нажатии на кнопку "Задача 1: Конвертер систем счисления" открывается форма для ввода числа. После ввода числа и нажатия кнопки "Преобразовать" программа выводит его представление в двоичном, восьмеричном и шестнадцатеричном виде.
- При нажатии на кнопку "Задача 2: Сложение в двоичной системе" пользователь вводит два двоичных числа. Калькулятор проверяет корректность ввода и выводит результат сложения в двоичном и десятичном форматах.
- При нажатии на кнопку "Задача 3: Визуализация двоичного числа" пользователь вводит десятичное число. В области PictureBox отрисовывается его двоичное представление: черные квадраты для '1' и белые с рамкой для '0'.

### Особенности программы
- Полная валидация ввода (проверка на число, пустой ввод, отрицательные числа, допустимые символы)
- Графический интерфейс с главным меню
- Визуальное отображение двоичного кода с помощью класса Graphics
- Разделение логики и представления с помощью статических классов-утилит

### Требования
- .NET 8.0 (или .NET 6.0/8.0+, в зависимости от версии, указанной в файлах)
- Visual Studio 2022 (или любая среда с поддержкой C#. Версия языка C# 10 и выше)
